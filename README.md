# Recap

**Recap** is a personal study tool designed to help you easily create, customize, and review your own study notes. It supports flexible note editing, flashcards, and multiple-choice quizzes to make learning more efficient and interactive. ([github.com](https://github.com/amiddle2/Recap.git))

---

## Features

- **Customizable Notes** – Create and edit notes in a way that fits your study style.
- **Flexible Review Modes** – Use notes as standard content, flashcards, or quiz questions.
- **Quiz Mode** – Test your knowledge with built-in multiple-choice quizzes.
- **Track Progress** *(future)* – Potential to expand with user progress tracking and analytics.

---

## Usage

1. **Create Notes**  
   Add your study material or definitions as note entries.

2. **Customize Study Style**  
   Choose between standard notes, flashcards, or quiz formats.

3. **Quiz Yourself**  
   Start a quiz session to reinforce learning.
